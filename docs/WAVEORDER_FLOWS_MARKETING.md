# WaveOrder Flows — Marketing & Sales Features

**Version:** 1.0  
**Date:** March 2026  
**Purpose:** Features available for any business type — for sales, marketing, and adoption messaging

---

## Overview

WaveOrder Flows is a WhatsApp messaging automation module built into WaveOrder. It works for **all business types**: Restaurant, Cafe, Retail, Grocery, Salon, Services, and Other. Features adapt to each type (e.g., "Orders" vs "Bookings" vs "Requests").

---

## Core Features (All Business Types)

### 1. WhatsApp Inbox

| Feature | Benefit |
|---------|---------|
| **Unified inbox** | See all customer WhatsApp conversations in one place — no separate app |
| **Reply from admin** | Answer customers directly from WaveOrder; no switching between tools |
| **Conversation history** | Full thread history per customer for context |
| **Unread badges** | Never miss a message; unread count in sidebar |
| **Mobile-responsive** | Reply from anywhere — inbox works on phone and tablet |
| **Status tracking** | Track conversations: open, assigned, waiting, resolved, closed |

### 2. Automated Welcome & Away

| Feature | Benefit |
|---------|---------|
| **Welcome message** | Auto-send a greeting with your store/catalog link when customers first message |
| **Away message** | When closed, automatically reply with opening hours and a link for later |
| **Business hours aware** | Flows respect your configured hours and timezone |

### 3. Keyword & Button Flows

| Feature | Benefit |
|---------|---------|
| **Keyword triggers** | Customers type "menu", "hours", "book" — get instant automated replies |
| **Button replies** | "Order Now", "Location", "Help" — one tap, instant response |
| **Pre-built templates** | Templates tailored to your business type (restaurant, salon, services, etc.) |
| **Custom flows** | Create your own flows with text, images, links, location, team notifications |
| **Notify team** | Alert your team when certain flows trigger (e.g., complaint, urgent request) |
| **Flow analytics** | See trigger count and last triggered per flow — know which flows perform best |

### 4. Visual Flow Builder

| Feature | Benefit |
|---------|---------|
| **Drag-and-drop** | Build flows visually with nodes and connections |
| **Condition & delay** | Add logic and timing to flows |
| **No coding** | Simple, intuitive interface |

### 5. AI Auto-Replies

| Feature | Benefit |
|---------|---------|
| **Intent detection** | AI understands product inquiries, FAQs, order status, complaints |
| **Instant answers** | When no flow matches, AI can reply using your products and FAQs |
| **FAQ training** | Train AI with your own Q&A pairs for personalized, accurate replies |
| **Human fallback** | Low-confidence replies route to your team |
| **Configurable** | Set personality (formal/friendly), confidence threshold, daily limit |

### 6. Team & Multi-Agent

| Feature | Benefit |
|---------|---------|
| **Assign conversations** | Route chats to specific team members |
| **Internal notes** | Add private notes visible only to your team |
| **Canned responses** | Quick-insert common replies |
| **Auto-assign** | New conversations distributed round-robin |
| **Agent presence** | See who's online |
| **SLA alerts** | Get notified when customers wait too long |
| **Supervisor view** | Filter conversations: All, Mine, or Unassigned |
| **Agent metrics** | Per-agent stats: assigned, resolved, resolution %, avg first-response time |

### 7. Broadcast & Campaigns

| Feature | Benefit |
|---------|---------|
| **Contact list** | Import from existing conversations or CSV |
| **Segments** | Target by tags or recent orders/bookings/requests |
| **Campaigns** | Send promotional messages (e.g., Valentine's Day offer, new menu) |
| **Save as draft** | Prepare campaigns in advance; send when ready |
| **Template variables** | Personalize with customer name, phone |
| **CSV template** | Download example CSV for easy contact import |
| **Opt-out handling** | Customers can reply STOP; automatically excluded from future campaigns |

## Use Cases by Business Type

### Restaurant

- Welcome & menu link when customers first message  
- Outside hours reply with opening times and order link  
- Keywords: menu, delivery, hours, specials  
- Daily specials flow when customers ask "what's new"  

### Cafe

- Welcome & order link for grab-and-go  
- Outside hours with pre-order link  
- Keywords: menu, coffee, hours, promo  
- Loyalty & promotions flow  

### Retail

- Welcome & catalog link  
- Outside hours with browse link  
- Keywords: catalog, delivery, prices, new arrivals  
- New arrivals & sales flow  

### Grocery

- Welcome & order link for weekly shopping  
- Outside hours with delivery info  
- Keywords: order, delivery, minimum, delivery area  
- Delivery zones & fees flow  

### Salon

- Welcome & booking link  
- Outside hours with book-online link  
- Keywords: book, appointment, services, prices  
- Service info & special offers flow  

### Services

- Welcome & request/quote form link  
- Outside hours with response time and request link  
- Keywords: quote, services, contact  
- Service areas & process flow  

### Other

- Generic welcome, away, catalog flows  
- Custom keyword flows for any use case

8. **Flow analytics** — See which flows get used most; optimize based on data
