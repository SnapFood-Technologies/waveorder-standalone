# Product video — recording prep

Internal prep for WaveOrder **marketing / product** videos (not feature implementation docs).

## Demo business type: **RETAIL**

All screen recordings in this series assume a **RETAIL** store (product catalog, cart, checkout — not salon appointments or service-request flows).

- **Include in admin demos:** Products, categories, variants/collections if the demo store uses them, **Inventory** (Pro), orders, customers, analytics, **Appearance**.
- **Do not anchor the series on:** Appointments calendar, service lists as primary catalog, service-request inbox (those are for **SALON** / **SERVICES** demos).
