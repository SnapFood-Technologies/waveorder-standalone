## Notes for Marketing Team

1. **Placeholders with variables**: Some keys contain placeholders like `{count}`, `{itemWord}`, `{inCart}`, `{available}`, `{stock}`. These should be preserved in translations as they will be replaced with actual values at runtime.

2. **Country codes**: Country code translations are handled programmatically (AL → Shqipëri, XK → Kosovë, MK → Maqedonia e Veriut). No manual translation needed for country codes.

3. **Format examples**: Keys like `phoneFormatAlbania`, `phoneFormatGreece`, etc., contain phone number format examples. The format structure should be preserved, only the word "Format" needs translation.

4. **Status messages**: Some messages may appear with status-specific context. Ensure translations maintain the meaning and tone appropriate for the context.

5. **Special characters**: Albanian translations include special characters (ë, ç, etc.). Please ensure these are correctly encoded in UTF-8.

6. **Consistency**: When translating, maintain consistency with:
   - Business terminology (e.g., "Porosi" for "Order")
   - Tone (formal/informal) - currently using formal "juaj/tënde" forms
   - Brand names (e.g., "WhatsApp" should remain as is)
